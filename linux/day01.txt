linux 使用

linux 命令格式：
    命令名　[选项]　[参数]

    1、who 登录到系统中的用户信息

	2、pwd 查看当前工作的目录
		/home/tarena
		1、根 /:文件系统的起始位置。
		2、路径：代表的是文件或者目录的字符串

	3、ls
		1、用于显示指定目录的文件或文件夹信息。
		2、ls [选项] [参数]
		3、选项：
		  -a 显示文件夹中的所有数据
		     （包含被隐藏的数据）
			.  代表当前的目录
			..  代表上一级目录
		  -l  以列表的形式显示文件的详细信息

	4、路径：
        路径是用来记录一个文件或文件夹位置的字符串
		1、绝对路径：以‘/’字符开始的路径为绝对路径
			/home/tarena
		ls /home/tarena/anaconda3/bin
		2、相对路径：不是以‘/’字符开始的路径
            开始符号有：
                文件／文件夹名
                . 当前文件夹
                .. 上一级文件夹
                ～　用户主目录
			anaconda3
			ls anaconda3/bin/

	5、cd
		1、作用：改变当前工作路径
		    格式：cd  [目录名]

			cd -:最近访问的两个目录中来回切换
			cd ~:回到家目录（主目录）
			cd /：回到根目录
            cd  : 默认不写目录名相当于　cd ~


	６、mkdir
		1、创建一个或者多个文件夹
		2、格式：mkdir [选项] 文件1  文件2...
			-p 如果中间文件夹不存在，　则创建中间所有文件夹

    ７、touch
	    1、作用：如果文件不存在，创建新的文件
	         如果文件存在，更改文件创建的时间

    ８、rmdir
	    作用：删除文件夹（空白的文件夹）

    ９、rm
	   １、 作用：删除文件或文件夹
       ２、格式： rm [ 选项]　文件／文件夹
	        -i :删除前提示（y同意 n 不同意）
	        -r :递归删除文件夹内部的文件夹和文件
	        -f :强制删除，不给提示信息。

    １０、clear:(ctrl+l):清屏
        ctrl+shift+"+":字体扩大
        ctrl+"-"：字体缩小
        init 0:关机
        init 6：重启
        ctrl+s:隐藏终端输出
        ctrl+q:恢复终端输出
        ctrl+c:终止命令的执行、

    １１、通配符。
	    ？ ：匹配一个任意字符。
	    * ：匹配０个，　１个，　多个任意字符。

    １２、Help 选项
        用于显示命令的帮助信息
        示例：
            ls --help

    １３、man 命令帮助
            格式：
                man linux/Unix 命令
            示例：
            　man ls
        退出键：　ｑ　
        Ｔａｂ键　命令行自动补全





    文件管理相关的命令

    １、vi vim编辑器：
	    1. vi  文件名
	    2、工作模式：
	    2.1 浏览模式
	    2.2 插入模式，在此模式下进行编辑文件
	    2.3 命令行模式
	    3、模式之间切换。
	        浏览模式--》插入模式	a i o
	        插入模式--》浏览模式  ESC
	        浏览模式--》命令行模式 shift +":"
	         :q 退出
	        ：w 写入
	        ：wq 
	        :q！ 强制退出

    ２、cp
	    1、作用复制文件或者目录
	    2、cp [选项]]　文件1（源文件或文件夹名） 文件2（目标文件或文件夹名）
            cp day01.txt ~/桌面/
	        cp 目录1 目录2 -r
        常用选项：
            －ａ 可以复制文件夹
        
    ３、mv 
	    1、文件移位置　或　更改名字
        格式：
	        mv 源文件或文件夹名　目标文件或文件夹名
	    mv 目录1 目录2
	    mv 路径/目录1 路径/目录2
    
    ４、cat 
	    文件中的内容输出到终端上。

    ５、more
	    作用：分屏查看文件内容。
	    回车：下一行
	    空格键：翻到下一屏
	    q:退出

    ６、head 
	    输出文件头几行的内容。
	    head 文件名（默认输出的前10行的内容）
	    head -n 文件名

    ７、tail 
	    输出文件尾几行的内容。
	    tail 文件名（默认输出后10行的内容）
	    tail -n 文件名

    ８、find
	        查找文件或者目录，结果是文件或者目录所在的路径信息。
        格式：
	        find  路径 -name "文件名/目录名"
	        find AID1808 -name "passwd_new1"

    ９、grep
	    在文件中查找指定的内容，及文本信息。
	        grep  "查找内容" [选项] 文件名或路径
        常用选项：
            －ｎ　显示行号
            －r  递归搜索文件夹内的文件

            grep "2222222" *
	        grep "tarena" passwd_new1
	 

    １０、wc 统计文件有多少行
	    wc -l  文件名

    １１、echo 显示文本
	    echo "要显示的字符信息"

    １２、输出重定向 >：
	    作用：把输出到终端上的数据，输出到指定文件中
	    ls -l > a.txt

    13、　gzip
        用ｚｉｐ压缩算法对文件　进行压缩，生成压缩后的文件后缀名通常用　.gz结尾
            命令格式：
                gzip　文件名
            gunzip 命令
                对用zip压缩算法对压缩的　.gz格式的数据进行解压缩
            示例：
                gzip day01.txt
                gunzip day01.txt.gz

    14、 tar
        打包解包的命令
        　对文件或文件夹进行打包和解包操作
        格式：
            tar [选项]　文件名或路径
        常用选项：
            －c         创建包
            -x          解包
            -f 文件名    操作的文件
            -v          显示操作的文件细节
            －z         用gzip/gunzip对包进行压缩和解压缩

            tar -czvf ~/day01.tar.gz *　＃打包
            tar -xzvf ~/day01.tar.gz 　　＃解包到当前目录

    15、sudo
        用超级用户root　权限来执行这个命令
        格式：
            sudo  命令　[选项]　［参数］
        常用选项:
            -i 选项　切换到root用户
        如:
            sudo -i     #进入root用户
